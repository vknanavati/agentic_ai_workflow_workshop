# Building and Scaling Agentic AI Workshops

A comprehensive workshop series covering three different approaches to implementing agentic AI workflows using AWS services.

## Workshop Structure

### Lab 1 - Building agentic workflows with AWS Step Functions
Learn how to implement tool use (function calling) with large language models and orchestrate workflows using AWS Step Functions.

**Modules:**
- Introduction to Agentic AI
- Understanding Tool Use
- Understanding Step Functions
- Building a Social Media Agent

### Lab 2 - Building and Deploying Agents with Strands
Explore code-first agent development using the Strands Agents SDK on AWS Lambda.

**Modules:**
- Creating Your First Agent
- Integrating External APIs
- Memory Management
- Retrieval Augmented Generation (RAG)
- Model Context Protocol (MCP)
- API Exposure

### Lab 3 - Amazon Bedrock Agents
Learn to build managed agents using Amazon Bedrock's native agent capabilities.

**Modules:**
- Creating Bedrock Agents
- Agent Orchestration
- Building Agents with Infrastructure as Code

## Prerequisites

- AWS account with appropriate permissions
- Familiarity with AWS services (Lambda, Step Functions, Bedrock)
- Basic understanding of AI/ML concepts

## Getting Started

1. Clone this repository
2. Navigate to the desired lab folder
3. Follow the module instructions in order
4. Complete the hands-on exercises

## Learning Objectives

After completing these workshops, you will be able to:
- Implement tool use with Amazon Bedrock models
- Build orchestrated workflows using AWS Step Functions
- Create code-first agents with Strands SDK
- Deploy managed agents with Amazon Bedrock
- Integrate agents with external APIs and data sources
- Implement memory and conversation state management
- Apply Retrieval Augmented Generation (RAG) patterns
- Build multi-agent systems

## Technologies Used

- Amazon Bedrock
- AWS Lambda
- AWS Step Functions
- Amazon API Gateway
- Amazon S3
- AWS CloudFormation
- Strands Agents SDK

## Clean Up

After completing the workshops, follow the clean-up instructions in each lab to avoid unnecessary costs.
