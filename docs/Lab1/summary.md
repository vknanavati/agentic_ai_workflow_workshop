# Summary

In this module, you learned how to implement the powerful pattern of tool use (function calling) with large language models (LLMs). You explored fundamental concepts including:

Tool Definition and Invocation: How to define tools using JSON Schema and make them available to foundation models
The Agent Loop Pattern: Implementing continuous reasoning cycles where models decide when to use tools and how to incorporate their results
Orchestration Approaches: Comparing code-based orchestration with workflow-based orchestration using AWS Step Functions
State Management: Handling conversation context and maintaining coherent interactions across multiple tool invocations
These core concepts can be applied to a wide range of use cases beyond the example we built. The pattern of having an LLM dynamically select and use tools based on context enables powerful automation capabilities across domains like content creation, data analysis, customer service, and more.
