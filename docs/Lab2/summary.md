# Summary

In this module, you learned how to build and deploy production-ready AI agents using Strands on AWS Lambda. You explored fundamental concepts including:

Agent Creation and Deployment: How to create a basic agent with local tools and deploy it to AWS Lambda for serverless execution
External API Integration: Connecting agents to external APIs to extend their capabilities beyond local tools
Tool and Model Configuration: Providing agents with access to multiple tools and foundation models, and customizing behavior through system prompts
Memory Management: Implementing persistent memory using external data stores like Amazon S3 to maintain context across conversations
Retrieval Augmented Generation (RAG): Leveraging Amazon Bedrock Knowledge Bases to enable agents to access and reason over private data
Model Context Protocol (MCP): Connecting agents to external MCP servers and dynamically loading remote tools using Amazon Bedrock AgentCore Gateway with OAuth2 authentication
API Exposure: Exposing agents through Amazon API Gateway with Amazon Cognito authentication for secure HTTP-based access
These core concepts enable you to build sophisticated AI agents that can be integrated into web applications, enterprise systems, and automated workflows across various domains.
